Usage:

1. After installing CLIPS, navigate to the directory containing the Dysmenorrhea Diagnosis System files.

2. Load the CLIPS rule and data files using the CLIPS command-line tool. Use the following commands:

   ```shell
   clips -f DysmenorrheaDiagnosisMain.clp
   clips -f DysmenorrheaDiagnosisData.clp
   ```

3. Run the system using the CLIPS environment with the following command:

   ```shell
   (reset)
   (run)
   ```
